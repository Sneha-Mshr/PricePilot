# PricePilot 🚀

PricePilot is an AI-powered price comparison system that helps users compare product prices across multiple e-commerce platforms such as Amazon, Flipkart, Myntra, Ajio, and Meesho.

## Objective

Instead of manually checking multiple websites, users can search for a product once and receive:

- Product prices from multiple platforms
- AI-powered recommendation
- Best deal available
- Savings analysis

## Tech Stack

### Backend
- FastAPI
- Python

### Frontend
- React
- Tailwind CSS

### AI
- RAG (Retrieval-Augmented Generation)
- Vector Database
- LLM
