# Problem Statement

## Existing Problem

When users want to buy a product, they usually visit multiple e-commerce websites to compare prices manually.

This process is:
- Time-consuming
- Repetitive
- Confusing due to different offers

## Proposed Solution

PricePilot is an AI-powered price comparison system that retrieves product information from multiple platforms and provides users with:

- Price comparison
- Best deal recommendation
- Savings analysis
- AI-generated summary

## Goal

Build an end-to-end RAG-based application using modern backend architecture and AI technologies.
